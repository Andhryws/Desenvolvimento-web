Por que Java é uma linguagem 'dificil'?


Java é uma linguagem de programação muito utilizada, mas para muitas pessoas, especialmente as que estão começando, pode achar difícil de aprender.
Isso acontece por algumas razões, como a forma como se escreve o código, a maneira como os tipos de dados são usados e a quantidade de coisas novas
para aprender. Além disso, muitas pessoas nem tentam aprender Java porque têm medo de não conseguir.

Motivos para java ser uma linguagem dificil

  • Escreer códigos complexos
  • Tipos de dados específicos da linguagem
  • Medo de não conseguir
